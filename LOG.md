## 2026-05-10 — エッジケースとリングポジション概念を導入

- **解決順序**: スピード差 + 確率（speed差 +5 で +25% 補正、0.1〜0.9 でクランプ）
- **回避失敗時**: ダッキング/スウェーが回避失敗したら **動かない**（被弾＋距離変化なし）
- **両者の動き累積**: 両者ダッキング成功 → 距離が2段近づく等、独立に適用
- **オフェンスは常に発動**: ディフェンスとは独立した枠。カウンター中でも自分のオフェンスは普通に出る
- **両者カウンター**: それぞれのオフェンス vs 相手カウンター でカウンター判定が独立に発生
- **リングポジション 概念導入** (新規):
    - 距離（相対）とは別に、各プレイヤーのリング上の絶対位置を管理
    - 中央 → ロープ際 → コーナー際 の3段階
    - 距離がクリンチまで詰まった状態でさらに押されると、押された側のポジションが後退
    - ロープ際/コーナー際では命中・回避にペナルティ
- `03_戦闘システム.md` を全面更新（§2.5 §2.6 §2.7）

    - スウェー: 回避（HEAD系の方が避けやすい）+ 距離広がる
    - カウンター: どちらかの攻撃が当たる（成功で1.3倍返し / 失敗で1.3倍被弾）

## 2026-05-10 — 戦闘システム第1ラウンド決定（コマンド体系・距離・ガード・カウンター）

- **オフェンス8種**: ジャブ/ストレート/フック/アッパー × HEAD/BODY
    - 威力順: ジャブ < ストレート < フック < アッパー
    - 距離適性: ジャブ=全距離、ストレート=遠、フック=中、アッパー=近
- **ディフェンス5種**: ダッキング / スウェー / ガード上 / ガード下 / カウンター
    - ダッキング: HEAD/BODY両方回避（HEADの方が避けやすい）+ 距離縮む
    - スウェー: HEAD/BODY両方回避（HEADの方が避けやすい）+ 距離広がる
    - ガード上/下: 部位読みが当たれば被ダメ0、外れれば通常ダメージ（クリーンボーナス無し）
    - カウンター: BODYの方が成功しやすい。成功で1.3倍返し、失敗で1.3倍被弾
- **距離 6段階**: クリンチ / 近 / 中 / 遠 / 離れ / コーナー際
- マトリクスは **3レイヤー（A: 部位 × B: 距離 × C: 防御技）の積** で計算
- これに伴い `03_戦闘システム.md` を全面書き直し
