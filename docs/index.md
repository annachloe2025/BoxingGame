# BoxingGame

「Boxing Arena Nebura」風の見た目に、テキストアドベンチャーの実況・シチュエーション描写を融合した、
1対1のターン制ボクシングゲーム企画。

> コンセプト：**ビジュアルは派手に、戦闘の中身はテキストで深く。**

## 現在のステータス：Phase 0（設計のみ）

設計書は **章立てと既決事項だけ** が入った初版で、これからユーザーと議論しながら埋めていく。
