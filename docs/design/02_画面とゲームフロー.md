# 02 画面とゲームフロー

## 1. 画面レイアウト

参考画像（Boxing Arena Nebura）と同じ骨格を踏襲。

```
┌────────────────────────────────────────────────────────────┐
│ ① 上部HUD: HEAD/BODY HPバー × 2、名前、ROUND、タイマー        │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  ② 左立ち絵        ③ 中央パネル        ④ 右立ち絵           │
│  (プレイヤー)      ┌─────────────┐      (敵)                │
│                    │ コマンド or  │                          │
│                    │ アクション   │                          │
│                    └─────────────┘                          │
│                                                            │
├────────────────────────────────────────────────────────────┤
│ ⑤ テキスト欄（アクション再生中だけ出現する1〜2行ウィンドウ）  │
├────────────────────────────────────────────────────────────┤
│ ⑥ 状態行: 距離 [中距離]    ⑦ サイドメニュー: タオル/オート他 │
└────────────────────────────────────────────────────────────┘
```

### ① 上部HUD

- **HEAD HP** / **BODY HP** の2本ずつ × 左右
- セグメント式（参考画像と同じ）：HEAD は赤系グラデ、BODY は緑系グラデ で識別
- 名前、VS マーク、ROUND 表示、3:00 タイマー（ラウンド制を採用するなら）

### ② / ④ 立ち絵

- 上半身〜腰くらいまで、画面の左右に大きく配置
- 状態に応じてポーズを差し替え：`neutral / guard / hit / down`
- 表情差分（クリーンヒット時に苦痛、クリンチ中はにらみ合い等）は v0.2 以降

### ③ 中央パネル（モード切替）

#### コマンド選択モード

```
┌────────────┐
│ OFFENCE 1   │
│ [ストレート]│
│             │
│ DEFENSE     │
│ [ガード]    │
│             │
│   [決定]    │
└────────────┘
```

- v0 は **オフェンス1 ＋ ディフェンス1** の2スロット
- 後の拡張で **オフェンス2** が増える前提のスロット設計（CSSグリッドで簡単に増やせる）

#### アクション再生モード

```
┌──────────────────────────────────┐
│ OFFENCE 1: ストレート            │
│ ┌────────┐  ╱╲╲  ┌────────┐      │
│ │ player │ ╱⚡╲ │ enemy  │      │
│ │ cut-in │ ╲⚡╱ │ cut-in │      │
│ └────────┘  ╲╱  └────────┘      │
│ DEFENSE: ガード                  │
└──────────────────────────────────┘
```

- 中央にカットイン2枚（プレイヤー / 敵の中サイズ画像）
- 上下に「OFFENCE 1: 〜」「DEFENSE: 〜」を表示
- 再生中はテキスト欄（⑤）が下に出現

### ⑤ テキスト欄

- アクション再生中のみ表示
- 1〜2行の薄い半透明ウィンドウ
- 種別を色分け：**実況**（黄）/ **台詞**（白／キャラ色）/ **システム**（灰）
- クリック or Enter で次へ

### ⑥ 状態行

- 距離感（クリンチ／近距離／中距離／離れ気味／ロープ際／コーナー）を1単語で
- スタミナ・必殺技ゲージなどがあればここに並べる

### ⑦ サイドメニュー

- **タオル**: 降参（v0 は動作未実装でOK）
- **オート**: AI 同士で自動進行
- **ステータス**: 詳細ステ表示
- **オプション**: 音量・テキスト速度

## 2. ゲームフロー

```
[タイトル画面] (v0は省略)
   ↓
[試合開始演出] ← ROUND 1 ゴング
   ↓
┌→ [コマンド選択モード] ← プレイヤー入力
│      ↓ 決定
│  [アクション再生モード] ← 自動進行
│      ↓ 全アクション終了
│  [HP/距離/状態 反映]
│      ↓
│  KO?  ── No → ↑ ループ
│      Yes
│      ↓
└─ [試合結果画面] (v0は KO 表示で終了)
```

## 3. ターンの内部処理（時系列）

1. ユーザーがコマンド選択（オフェンス1 ＋ ディフェンス）
2. 敵AI が同時に決定（プレイヤーには見えない）
3. **決定ボタン押下** → モード切替アニメ（中央パネルが変形）
4. 速度比較で先攻判定
5. 先攻のオフェンスを後攻のディフェンスにぶつける
   - 命中／クリティカル／空振り／ガード の判定
   - HP 反映
6. 後攻のオフェンスを先攻のディフェンスにぶつける（先攻でKO の場合スキップ）
7. テキスト欄で実況・台詞を順次表示（クリックで進める）
8. 距離・状態を更新
9. KO 判定 → 続行 or 終了
