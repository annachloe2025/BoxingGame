# 00 設計サマリー

> このページだけで「何を作ろうとしているのか」を1分で把握できることを目指す。
> 詳細は 01〜07 へ。

## ジャンル

**アドベンチャー型ボクシングゲーム。**
ビジュアルは「Boxing Arena Nebura」を強く参考にしつつ、戦闘の中身はテキストアドベンチャーの実況・シチュエーション描写で進める。

## 1試合の核ループ（v0）

1. **コマンド選択** ─ 中央パネルで オフェンス1 + ディフェンス1 を選んで「決定」
2. **アクション再生** ─ 両者のコマンドが噛み合って自動再生。テキスト欄に実況・台詞・距離変化が流れる
3. **状態反映** ─ HEAD / BODY のHPと距離感が更新される
4. ↑ を繰り返し、HPが尽きるか KO で試合終了

## 画面の骨格（最終形イメージ）

```
┌────────────────────────────────────────────────┐
│ [HEAD HP] ──────────  VS  ────────── [HEAD HP] │
│ [BODY HP] ──────────       ────────── [BODY HP] │
│             ROUND 1   3:00                      │
├────────────────────────────────────────────────┤
│                                                │
│  [大きな立ち絵]      [中央パネル]    [大きな立ち絵] │
│   (プレイヤー)     ┌──────────────┐   (敵)        │
│                   │ コマンド or   │                │
│                   │ アクション    │                │
│                   │ (モード切替)   │                │
│                   └──────────────┘                │
│                                                │
├────────────────────────────────────────────────┤
│ [テキスト欄] 「まどか、ここは身を固めるっ！」    ▼ │
├────────────────────────────────────────────────┤
│ 距離: 中距離   |  タオル | オート | ステータス   │
└────────────────────────────────────────────────┘
```

中央パネルは2モード：

- **コマンド選択モード**: オフェンス1 / ディフェンス / 決定ボタン
- **アクション再生モード**: 現在実行中のコマンド表示 + キャラのカットイン演出

## 既決事項（2026-05-10 時点）

- [x] **戦闘**: v0 は **オフェンス1 + ディフェンス1** の同時決定 → 自動再生
- [x] **HPバー**: 上部に **HEAD / BODY の2系統**（参考画像に合わせる）
- [x] **距離感**: テキスト中心で表現（「クリンチ」「中距離」「ロープ際」など）。チビキャラ表示は廃止寄り
- [x] **テキスト欄**: 常設ではなく、アクション再生中に画面下に1行ウィンドウとして表示
- [x] **プロジェクト構造**: ThinkingLab を踏襲した mkdocs + GitHub Pages
