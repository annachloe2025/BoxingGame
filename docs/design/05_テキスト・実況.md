# 05 テキスト・実況

このゲームの戦闘の **「中身」** はここ。テキストの密度と質がそのまま面白さになる、という前提で設計する。

## 1. テキストの3レイヤー

| 種別 | 例 | 表示色 | タイミング |
|---|---|---|---|
| **実況** | 「赤コーナー、鋭いジャブが入った！」 | 黄系 | アクションのたびに毎回 |
| **台詞** | 「（ここで決める……！）」 | キャラ色 / 白 | 重要な瞬間だけ（クリーンヒット、被弾、KO 寸前等） |
| **システム** | 「HEADに22ダメージ。残り78。」 | 灰 | デバッグ寄り、設定でON/OFF |

## 2. テンプレ駆動の実況

実況テキストは「(攻撃 × 結果 × 距離 × HP残量) → テンプレ群からランダム抽選」する形で量産する。

`commentary.json` のスキーマ案：

```json
{
  "JAB": {
    "hit": [
      "{atk}の鋭いジャブが顔面を捉えた！",
      "小気味よいジャブ！{def}の前進を止める！"
    ],
    "miss": [
      "{atk}のジャブはわずかに届かない！",
      "{def}が上体を引いて避ける！"
    ],
    "guard": [
      "{def}は腕でジャブを受け止めた。",
      "ガードの上から、ジャブが届く。"
    ],
    "crit": [
      "完璧なタイミングのジャブ！{def}の足が止まる！"
    ]
  },
  "STRAIGHT": { ... },
  "HOOK": { ... },
  "GUARD": { ... }
}
```

## 3. 台詞（キャラ別）

各キャラに `voice_set.json` を持たせ、シチュエーション別に複数のセリフを格納。

```json
{
  "player_01": {
    "round_start":  ["やってやる…！", "今日は絶対に勝つ！"],
    "land_clean":   ["これだ！", "効いてる…！"],
    "got_hit":      ["くっ…", "（こいつ、効くな）"],
    "near_ko":      ["（まだだ…まだ立てる…）"],
    "ko_win":       ["……勝った…", "オレの勝ちだ！"],
    "ko_lose":      ["……ちくしょう…", "次は…必ず…"]
  }
}
```

> **TBD**: シチュエーションの粒度。最初はもっと粗くて良いはず。

## 4. 距離感の表現

**画面上で1単語の状態表示** ＋ **テキスト欄での描写** の二段構え。

例：
- 状態行: `距離: クリンチ`
- テキスト欄:
    - 「両者組み合った。レフェリーがブレイクをかける。」
    - 「至近距離で短打の応酬。」
    - 「{atk}が肩で押して距離を作った。」

> **TBD**: 距離の遷移をテキストで描写するためのテンプレ群。

## 5. 速度・テンポ

- **テキスト送り**: クリック or Enter で次へ。**自動再生モード**（一定速度で流れる）も用意。
- **重要シーン**は通常の半分の速度で表示する（クリーンヒット・KO 寸前など）。
- **連続ヒット**は1メッセージにまとめて速度を上げる（手数で押す表現）。

## 6. 多言語対応

v0 は **日本語のみ**。
将来は `i18n/ja.json` `i18n/en.json` のように分離可能な構造で書いておく。
