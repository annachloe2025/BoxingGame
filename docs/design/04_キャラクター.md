## 3. ステータス構造（2026-05-10 確定: A路線=詳細）

「Boxing Arena Nebura」風の細かい数値構造を採用。
キャラの個性が出やすい代わりに、調整パラメータが多い。

### 3.1. PHYSICAL（基礎能力）7項目

| 項目 | 英語名 | 範囲 | 用途 |
|---|---|---|---|
| 顔面HP | head_hp | 12〜20 | HEAD HPの最大値（ダウン判定の片方） |
| ボディHP | body_hp | 12〜20 | BODY HPの最大値（ダウン判定のもう片方） |
| パンチ力 | punch_power | 10〜20 | 全攻撃の威力ベース倍率 |
| 耐力 | endurance | 10〜20 | 被ダメ全般軽減（chin/gut の元値） |
| スタミナ | stamina | 10〜20 | スタミナ最大値とターン消費耐性 |
| タフネス | toughness | 10〜20 | ダウン後の **立ち上がり率** に効く |
| カウンター | counter | 10〜20 | カウンターディフェンスの **成功率** に効く |

### 3.2. PUNCH（技ごと攻撃/防御）8項目

技の **熟練度**。攻撃は命中・威力に、防御は被ダメ軽減に効く。

| 項目 | 用途 |
|---|---|
| ジャブ攻撃 / ジャブ防御 | jab_atk / jab_def |
| ストレート攻撃 / ストレート防御 | straight_atk / straight_def |
| フック攻撃 / フック防御 | hook_atk / hook_def |
| アッパー攻撃 / アッパー防御 | upper_atk / upper_def |

> 技の熟練度は HEAD/BODY 共通（「ジャブ熟練度」が高ければ HEADジャブ・BODYジャブの両方に効く）

### 3.3. SPECIAL（個性）

- **スキル**（パッシブ）: 1〜3個。例: 「クール」「観察眼」「リズム支配」「衝突」など
- **必殺フロー**（特殊技）: 1個、固定名。**v0 では使用しない**（Phase 3 で実装）

```yaml

  id: 1
```

> 数値は **仮**。Phase 1 のプロトでバランス調整しながら埋める。

### 3.5. 評価値（総合）

サンプル画面に従って、参考に **総合評価値** を出す式：

```
total_eval = sum(physical) * 5 + sum(punch) * 3
           # 例: physical 7項目 × 平均15 = 105 → ×5 = 525
           # punch 8項目 × 平均14 = 112 → ×3 = 336
           # total ≒ 861 ... → 2300〜2400 にスケーリング: ×3 して 2580 など
```

> ライセンス（E〜A〜S）はこの評価値で決まる、という設計。詳細はTBD。

## 4. キャラ別の数値方針（叩き台）

各キャラのスタイルに合わせて **強み/弱み** を数値で表現する目安：

| キャラ | 強み | 弱み |
|---|---|---|
| エリー（アウトボクサー） | jab_atk高 / stamina高 / speed高 / 距離管理系 | upper_atk低 / power低 |
| 黒峰サツキ（カウンター） | counter高 / 全防御系高 / 観察 | スタミナ普通 / 連戦に弱い |
| エマ（インファイト） | body攻撃熟練度高 / upper_atk高 / endurance高 | jab_atk中 / 読み合いに弱い |
| 赤城ナギサ（ハードパンチャー） | punch_power高 / hook_atk・upper_atk高 / power系 | スタミナ低 / 防御熟練度低 |
| 桜野アイリ（コントロール） | 防御熟練度高 / counter高 / tech高 | punch_power低 |
| 橙川ヒカリ（ラッシュ） | speed高 / jab_atk高 / stamina高 | counter低 / 防御低 |
| ミラ（省エネアウト） | endurance高 / stamina高 / jab_atk・straight_atk高 | 火力低 |
| シオン（コントロール） | jab_atk高 / speed高 / 防御熟練度中 | プレッシャー耐性低 |
| リン（プレッシャー） | hook_atk高 / 進入系高 / 圧 | 読まれやすい |
| ハルカ（テクニック） | tech系 / 防御熟練度高 / counter高 | フィジカル低 |
