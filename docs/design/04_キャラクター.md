# 04 キャラクター

## 1. v0 のキャラ構成

最低限：

- プレイヤー側 1 人（プリセット）
- 敵 1 人（プリセット）

> **TBD**: 名前・性別・年齢・体格・性格などのパーソナリティ。01の「面白さの核」と「画風」が決まってから具体化する。

## 2. ステータス（共通スキーマ）

```yaml
Character:
  id: string                  # 例) "player_01"
  display_name: string        # 表示名
  age: int                    # 任意
  height: int                 # 任意 cm
  weight: int                 # 任意 kg
  personality: enum           # アグレッシブ / テクニシャン / カウンター / etc.
  art:
    neutral: path
    guard: path
    hit: path
    down: path
  stats:
    head_hp_max: int          # 例) 100
    body_hp_max: int          # 例) 100
    stamina_max: int          # 例) 100
    power: float              # 攻撃倍率
    speed: int                # 行動順・回避補正
    technique: int            # クリ・命中補正
    chin: int                 # 被ダメ軽減
  preferred_distance: enum    # クリンチ / 近 / 中 / 離れ
  voice_set: id               # 台詞セット参照
```

## 3. パーソナリティ（仮）

| パーソナリティ | 戦い方 | コマンド傾向 |
|---|---|---|
| アグレッシブ | 前進・連打 | フック多め、ガード少なめ |
| テクニシャン | 距離支配 | ジャブ・ダッキング多め |
| カウンター | 待ち | パリィ多め、被弾後にストレート |
| アウトボクサー | 離れて撃つ | ジャブで距離維持、近距離で逃げる |
| ハードパンチャー | 一撃必殺 | フック・アッパーに偏る |

> **TBD**: 5種類は多いかもしれない。v0 は 2〜3 種で十分か。
